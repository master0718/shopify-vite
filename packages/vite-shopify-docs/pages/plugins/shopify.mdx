[View this package on NPM.](https://www.npmjs.com/package/vite-plugin-shopify)

The core of this project is `vite-plugin-shopify`, which allows Shopify theme developers to use Vite as their front-end build tool. This plugin has everything you need to take advantage of all the great features of Vite when developing Shopify themes. More details can be found in the plugin README [here](https://github.com/barrel/barrel-shopify/blob/main/packages/vite-plugin-shopify/README.md).
